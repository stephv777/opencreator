# Decision 0003 — One Project, One Container

Status: Accepted

## Context

Creative work usually consists of many different files.

Text.
Images.
Audio.
Video.
Notes.
Versions.
Metadata.

These files should never become separated.

## Decision

Every creation belongs to a Project.
A Project contains everything related to that creation.

## Consequences

Nothing gets lost.
Projects become portable.
Backup becomes straightforward.

Future software can still understand the project structure.
