# Preserve

Every creation deserves to be preserved.
OpenCreator automatically keeps together:

- content
- media
- metadata
- versions
- publication history

Nothing is scattered.
Nothing is hidden.

---

Preservation happens before publication.
A creator should never lose work because an online service disappears.

---

OpenCreator can preserve projects through:

• local storage
• external drives
• network storage
• IPFS

Other preservation methods may be added over time.

---

Preservation is silent.

Automatic.

Reliable.
