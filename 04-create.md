# Create

Every creation begins with an intention.
OpenCreator does not ask:
"What software do you want to use?"
Instead, it asks:
"What would you like to create?"

---

Examples

○ A website
○ A book
○ A video
○ A podcast
○ A photo gallery
○ A course
○ A presentation
○ A document
○ A project

---

After choosing the intention, OpenCreator prepares the appropriate environment.
Whenever several Free/Open Source tools can achieve the same goal, OpenCreator selects the most appropriate according to the creator's needs.
The creator remains free to choose another tool at any time.

---

The workflow always follows the same philosophy:

Create
↓
Preview
↓
Improve (optional)
↓
Preserve
↓
Publish
---

The creator focuses on content.

OpenCreator handles formats, files and technical preparation.
