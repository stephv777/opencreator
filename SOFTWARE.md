# Software Selection

OpenCreator does not reinvent existing software.
It connects proven Free/Open Source applications into one coherent environment.
Whenever possible, at least two actively maintained solutions are evaluated for each major function.
The objective is not to promote one application.

The objective is to offer the best experience for creators.

| Need | Preferred Software | Alternative |
|------|--------------------|-------------|
| Website | Hugo | Publii |
| Image editing | GIMP | Krita |
| Video editing | Kdenlive | Shotcut |
| Audio editing | Audacity | Tenacity |
| Books | Calibre | Foliate |
| Notes | Joplin | Logseq |
| Synchronization | Syncthing | Nextcloud |
| Code hosting | Forgejo | GitHub |
| Video publishing | PeerTube | YouTube |
| Messaging | Matrix | XMPP |

Software choices may evolve.

The principles remain.
