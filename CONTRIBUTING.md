# Contributing

Thank you for your interest in OpenCreator.
OpenCreator welcomes every kind of contribution.
Programming is only one way to help.
---

You can contribute by:

- improving documentation
- translating texts
- correcting mistakes
- proposing ideas
- testing prototypes
- reporting bugs
- designing icons
- improving accessibility
- creating tutorials
- reviewing interfaces

---

## Our philosophy

We value respectful discussions.
We prefer thoughtful decisions over quick decisions.
We prioritize long-term sustainability over short-term trends.

---

Every contribution should answer one simple question:
Does this help creators?
If yes,
it probably belongs here.
