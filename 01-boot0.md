# Boot 0

Boot 0 is the beginning of every OpenCreator installation.
Its objective is simple:
Prepare everything automatically.
The creator should never have to understand Linux, partitions or package managers.

---

## Boot 0 checks

OpenCreator automatically checks:

- processor
- memory
- storage
- network
- Internet connection

---

## Boot 0 proposes

Welcome.
What would you like to do?

○ Discover OpenCreator
○ Try without installing
○ Install OpenCreator

---

If installation is selected:

OpenCreator:

- verifies compatibility
- prepares storage
- installs Debian
- installs OpenCreator Environment
- configures the basic services
- creates the first administrator
- reboots

Nothing else is required.

---

After reboot

The Dashboard appears.

The creator can immediately begin working.
