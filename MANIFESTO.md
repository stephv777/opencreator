# Manifesto

OpenCreator is built around one idea:

Creators should spend their time creating.

Technology should quietly take care of everything else.

## Our first milestone

OpenCreator 0.1 is not a complete operating system.

It is a working creator environment.

A creator should be able to install OpenCreator on Debian, create a first project, preserve it locally and publish it without technical knowledge.

Everything else will grow from this first experience.
